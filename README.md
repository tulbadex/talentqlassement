## Book Project

### To run this project
[*] Clone the repository
[*] Run composer update
[*] Generate new key
[*] Create a new database

### Endpoints
[*] To get external book
** POST /api/external-books?name=:name to search for
[*] To create a book
** POST /api/v1/books
[*] To get a book
** GET /api/v1/books/:id
[*] To update a book
** PATCH /api/v1/books/:id
[*] To delete a book
** DELETE /api/v1/books/:id
